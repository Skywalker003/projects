# Kalanjiyum Technical Solutions

A modern company website built with React and Vite.

## Tech Stack

- **React 19** with React Router v7
- **Vite 8** for bundling and dev server
- **Framer Motion** for animations
- **Lucide React** for icons

## Pages

| Route | Page |
|-------|------|
| `/` | Home |
| `/about` | About |
| `/services` | Services |
| `/portfolio` | Portfolio |
| `/careers` | Careers |
| `/careers/apply` | Job Application |
| `/contact` | Contact |
| `/terms-and-privacy` | Terms & Privacy |

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Project Structure

```
src/
├── components/
│   ├── layout/        # Navbar, Footer
│   ├── sections/      # Page sections (home, about, careers, contact)
│   └── ui/            # Reusable UI components
├── pages/             # Page-level components
├── data/              # Static data
├── hooks/             # Custom React hooks
└── assets/            # Images and static assets
```
